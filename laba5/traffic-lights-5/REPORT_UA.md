### 7. Налаштування маршрутизації

**Конфігурація React Router:**
- Використано `createBrowserRouter` для визначення маршрутів
- Layout компонента з Header та Outlet
- Обробка помилок через errorElement

**Код: `src/App.jsx`**

**Маршрути:**
- `/` - Home сторінка (index route)
- `/vertical` - Вертикальний світлофор
- `/horizontal` - Горизонтальний світлофор
- `*` - ErrorPage для неіснуючих маршрутів

**Опис:**
- Layout компонента обгортає всі сторінки
- Header відображається на всіх сторінках (крім ErrorPage)
- RouterProvider надає маршрутизацію додатку

## Результати роботи

### Реалізовані функції:

1. **Маршрутизація:**
   - Налаштовано React Router для навігації
   - Створено три основні маршрути
   - Обробка неіснуючих маршрутів через ErrorPage
   - Використано Layout для спільних елементів

2. **Компонента Header:**
   - Навігаційне меню з трьома пунктами
   - Активний стан для поточної сторінки
   - Фіксоване позиціонування
   - Адаптивний дизайн

3. **Сторінка Home:**
   - Інформація про лабораторну роботу
   - Опис мети та завдань
   - Список функціоналу
   - Стильний дизайн

4. **Сторінка ErrorPage:**
   - Кастомна обробка помилок
   - Відображення коду та опису помилки
   - Кнопка повернення на головну
   - Анімації та ефекти

5. **Сторінки світлофорів:**
   - Окремі сторінки для вертикального та горизонтального режимів
   - Незалежний стан для кожної сторінки
   - Збереження статистики при навігації

### Технічні деталі:

- **React Router v6:** createBrowserRouter, NavLink, Outlet, useRouteError
- **Layout Pattern:** Спільна структура з Header для всіх сторінок
- **State Management:** Локальний стан для кожної сторінки
- **Navigation:** Програмна навігація через Link та NavLink
- **Error Handling:** Кастомна сторінка помилок
- **CSS:** Модульна організація стилів для кожної компоненти
- **Responsive Design:** Адаптивність для різних розмірів екранів
